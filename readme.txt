Это руководство, которое трактует запуск и особенности проекта.

1. Технологии:
    Spring Boot
    Spring Data (Hibernate)
    Spring Security
    HTML/CSS(Bootstrap) + Thymeleaf with SpringSecurity5
    СУБД MYSQL

2. Функционал проекта:
    Проект представляет из себя блог, где есть возможность регистрации, аутенфикации и авторизации. Сущестует две роли:
    Админ и Юзер. Юзер имеет доступ к добавлении статьи, чтению статьи и просмотру своего профиля, где отображается
    информация о нем, кол-ве его публикаций, и посылания на его публикации.
    Админ имеет страницу, на которую поступают статьи юзером, он может их: принять, и статья появится на главной страни-
    це, отредактировать или удалить. Также у админа под каждой статьей есть интерфейс для удаления или редактирования
    статьи.

3. Запуск:
    В файле application.properties впишите данные для подключения к вашей базе данных MySQL. Таблицы по примерам Entity
    не нужно создавать, они сами создадутся при первом запуске приложения. Также в том же файле можно выставить порт,
    на котором будет доступ к сайту по локальному хосту.
    Чтобы стать админом, нужно создать отдельную учетную запись и изменить ей роль через СУБД
